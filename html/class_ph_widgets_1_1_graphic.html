Detailed Description

Common resources for graphical widgets.

The Graphic superclass provides the common resources used by all graphic widgets. Graphical widgets provide attributes for color, fills, patterns, line thickness, joins, and much more.

Remarks
When you want to incorporate simple, static drawings in your interface, use the subclasses of Graphic.
