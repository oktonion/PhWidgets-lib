Enumerator
Activate 

To activate a widget, you typically release the left pointer button while pointing at an armed widget.

A list of PtCallback_t structures that define the callbacks that the widget calls when it becomes activated.

Arm 

To arm a widget, you typically press the left pointer button while pointing at the widget.

A list of PtCallback_t structures that define the callbacks that the widget calls when it becomes armed.

Disarm 

To disarm a widget, you typically release the left pointer button when not pointing at an armed widget.

A list of PtCallback_t structures that define the callbacks that the widget calls when it becomes disarmed.

GotFocus 

(e.g. a child widget gets focus from its parent or the focus switches from a child to its parent).

A list of PtCallback_t structures that define the callbacks invoked when a widget gets focus or its focus status changes

LostFocus 

A list of PtCallback_t structures that define the callbacks that the widget calls when it loses focus.

Menu 

A list of PtCallback_t structures that define the callbacks that the widget calls when you press the right button while the pointer is on top of the widget.

Repeat 

These events occur when you hold down the left pointer button (or the right pointer button if the widget has Widget::Flags::AllButtons set in its Widget::Arguments<>::Flags resource).

A list of PtCallback_t structures that define the callbacks that the widget calls when it receives Ph_EV_BUT_REPEAT events.
