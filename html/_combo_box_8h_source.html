49  //! A text field with a list of choices
50  /*!
51  The ComboBox class provides a widget that's built from two exported subordinate widgets:
52  [List](@ref PhWidgets::List) and [Text](@ref PhWidgets::TextWidget).
53 
54  A ComboBox widget provides a text-entry area and a list of choices.
55  @par
56  You can type in the text field or choose a predefined entry from the list.
57  The list can be either:
58  @par
59  - Static: Always present above or below the text field.
60  - Dropping: You must click a button to see the list.
61  @par
62  The widget behaves like a List or Text widget, depending on which part has focus.
63 
64  @note
65  You can't specify the selection mode for the list.
66 
67  To select an item using the pointer, either click on an item or drag the pointer down the list
68  and release the button when the correct item is highlighted.
69  You can select only one item.
70  If you drag the pointer, the list can scroll.
71  @par
72  A blocking mechanism lets ComboBox block the inheritance of certain resources from its subordinate widgets.
73  This prevents any actions that would negatively affect the look and behavior of a ComboBox widget.
74  */
