enum PhWidgets::Widget::ThisArgs::ArgPChar::eArgPChar
Enumerator
help_topic 

The meaning of this resource depends on the bits set in Pt_ARG_EFLAGS: If Pt_INTERNAL_HELP isn't set, Widget::ArgPChar::help_topic is used to set the topic position within the HTML help file. If Pt_INTERNAL_HELP is set, Widget::ArgPChar::help_topic is the help information to be displayed.
