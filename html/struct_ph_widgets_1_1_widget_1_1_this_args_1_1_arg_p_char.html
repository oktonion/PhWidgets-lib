help_topic 

The meaning of this resource depends on the bits set in Widget::Flags::eflags: If Widget::Flags::Extended::internal_help isn't set, Widget::Flags::help_topic is used to set the topic position within the HTML help file. If Widget::Flags::Extended::internal_help is set, Widget::Flags::help_topic is the help information to be displayed.
