PhWidgets::Drawing::Color::B

Gets the blue component value of this Color class.

Property Value

PgColor_t : 8

The blue component value of this Color.

Remarks
The color of each pixel is represented as a 32-bit number: 8 bits each for alpha, red, green, and blue (ARGB). Each of the four components is a number from 0 through 255, with 0 representing no intensity and 255 representing full intensity. Likewise, Color::B is a value from 0 to 255 with 0 representing no blue and 255 representing fully blue.
