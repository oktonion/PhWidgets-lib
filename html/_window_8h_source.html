17  /*!
18  The Window class provides a top-level container for your applications' widgets.
19  It also provides a standard appearance for all windows.
20  Windows are managed by the Photon window manager if it's present.
21 
22  @note
23  A Window is the top-level Widget of the Application.
24  If you try to use another class for the top-level Widget (aside from a Region),

26 
27  ### Interacting with the Photon window manager ###
28

30  You can control which window management functions the window manager will perform,
31  and whether they're invoked from the window menu or from one of the window controls.
32  You can also have your application notified when the user has requested a window management function,
