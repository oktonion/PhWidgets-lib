Remarks
The dither pattern is an array of 8 bytes, aligned with the upper-left corner of the application's region. This pattern repeats itself every 8 pixels horizontally and every 8 pixels vertically.

Possible values:

  • Pg_PAT_DEFAULT
  • Pg_PAT_HALF
  • Pg_PAT_BACK_HALF
  • Pg_PAT_CHECKB8
  • Pg_PAT_CHECKB4
  • Pg_PAT_DIAMOND
  • Pg_PAT_HORIZ8
  • Pg_PAT_HORIZ4
  • Pg_PAT_HORIZ2
  • Pg_PAT_VERT8
  • Pg_PAT_VERT4
  • Pg_PAT_VERT2
