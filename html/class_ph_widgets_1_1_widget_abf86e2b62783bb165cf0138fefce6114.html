◆ Hide()

void PhWidgets::Widget::Hide ( )

Conceals the widget from the user.

Remarks
Hiding the widget is equivalent to setting the Widget::Visible property to false. After the Widget::Hide method is called, the Widget::Visible property returns a value of false until the Widget::Show method is called.
