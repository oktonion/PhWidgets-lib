190 
191  Raster Op Defines
192 
193  Reverse Polish Notation Defines of ROPS
194  This follows Microsofts 256 raster operation
195  definitions
196 
197  D = Dest Bit
198  S = Source Bit
199  P = Pattern Bit
200  o = bitwise or operation
201  a = bitwise and operation
202  x = bitwise exclusive or operation
203  n = bitwise not operation
204 
205  Truth Table
206  P S D ropbit
207  0 0 0 0
208  0 0 1 1
209  0 1 0 2
210  0 1 1 3
211  1 0 0 4
212  1 0 1 5
213  1 1 0 6
214  1 1 1 7
215 
216  if the condition in the table is true for
217  each test turned on then dst bit=1, else 0
218 
219  So: DrawMode CC = Source Copy
220  if ((conditions 2 | 3 | 6 | 7) == 1) on current bits, dst bit=1, else 0
221  DrawMode 66 = S ^ D
222  if ((conditions 1 | 2 | 5 | 6) == 1) on current bits, dst bit=1, else 0
223  DrawMode 03 = ~ (P | S)
224  if ((conditions 0 | 1) == 1) on current bits, dst bit=1, else 0
225 
226  It's a little confusing at first, but it does make sense once you wrap
227  your brain around it.
