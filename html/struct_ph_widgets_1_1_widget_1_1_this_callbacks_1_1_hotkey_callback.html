enum PhWidgets::Widget::ThisCallbacks::HotkeyCallback::eHotkeyCallback
Enumerator
hotkey 

A list of PtHotkeyCallback_t structures.

If the widget receives a key event that matches a structure's key cap and key modifiers, the widget calls the function specified in that structure. If a function isn't specified, the widget invokes its Basic::Callbacks::activate callback list with a reason_subtype of Widget::Callbacks::hotkey.
