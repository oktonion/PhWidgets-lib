phproperty<PgColor_t>::bind<Basic, ArgColor::eArgColor, Arguments::dark_bevel_color> PhWidgets::Basic::DarkBevelColor

Gets or sets the dark outermost color used when applying a bevel to a widget.

Property Value

typedef unsinged long PgColor_t

A PgColor_t, with Basic::LightBevelColor, that represents the outermost colors used when applying a bevel to a widget.

Note
These value is automatically generated when you set Basic::FillColor. Setting Basic::FillColor overrides any values set previously via these property.
Remarks
