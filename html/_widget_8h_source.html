659  //! Contains flags for Widget resource Widget::Arguments::resize_flags.
660  struct Resize
661  {
662  //! Resize flags for Widget resource Widget::Arguments::resize_flags.
663  /*!
664  Resize flags to control a widget's resize policy in both the x and y directions.
665 
666  @note
667  Each ..._bits flag is a mask that represents all the bits of that type.
668  The default setting of this resource is 0; that is, no resize policy is in effect.
669  A widget's resize policy deals solely with the widget's renderable data.
670  For a button, the data is its text; for a container, the data is its children.
671  @par
672  Any rendered data that doesn't fit within the widget's canvas is clipped.
673  If no resize policy is in effect, the widget's size is unbounded;
674  it may be made as large or small as specified via Widget::Arguments::dim or Widget::Arguments::area.
675  @par
676  If a resize policy is in effect, the widget grows or shrinks to honor that policy:
